# Vibecoding Amsterdam

Event site for vibecoding workshops at the Embassy of the Free Mind in Amsterdam.

**Live:** [vibecodingevents.vercel.app](https://vibecodingevents.vercel.app)

## What is vibecoding?

Vibecoding means describing what you want in plain language and watching it appear. A website, a game, a piece of art. If you can imagine it, you can probably build it.

## Tech stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000)

## Deploy

```bash
vercel --prod
```

## Feedback widget

This site includes a human-in-the-loop feedback widget. Add `?getinput` to any URL to enable review mode for external reviewers.
